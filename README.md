# 급여 자동 정산 프로그램

엑셀 근태 파일을 업로드하면 근무자별 급여 정산서와 전체 급여 요약을 자동 계산하는 웹앱입니다.

## 주요 기능
- 엑셀 업로드: `.xlsx / .xls / .csv`
- 근무자별 시급 설정
- 수당 적용 ON/OFF
  - 주휴수당
  - 공휴일수당(일요일 제외)
  - 연장근무수당(1일 8시간 초과분)
  - 야간수당(개인별 야간 시작시각, 전체 야간 종료시각)
- 누락 기록 보정
  - 출퇴근 누락 시 기본 출근시각/기본 근무시간 자동 적용
  - 누락 근무 수동 추가
- 근무시간 수동 보정
  - 예: `3.2시간 -> 4시간`
- 결과 화면
  - 근무자별 정산서(날짜/출퇴근/주차별 급여)
  - 전체 급여 요약(전 직원 총계/세후 급여)

## 계산 기준
- 기본급: `정산시간 × 시급`
- 연장/야간/공휴일 수당: 해당 시간에 대해 `추가 0.5배`를 가산
- 공휴일수당: 사용자가 등록한 공휴일만 적용, 일요일은 제외
- 주휴수당: 주간 근무시간이 기준시간 이상일 때
  - `주휴시간 = (주간 근무시간 / 주간 근무일수) × 주휴 가중치`
  - `주휴수당 = 주휴시간 × 시급`
- 세후급여: `급여총계 - (급여총계 × 세율)`

## 로컬 실행
정적 파일 앱이라 별도 빌드가 필요 없습니다.

1. 이 폴더에서 간단한 정적 서버 실행
   - PowerShell 예시: `python -m http.server 5500`
2. 브라우저에서 `http://localhost:5500` 접속

## GitHub + Render 배포
1. GitHub 저장소 생성 후 현재 파일 업로드
2. Render에서 `New +` -> `Static Site`
3. GitHub 저장소 연결
4. 설정
   - Build Command: 비워두기
   - Publish Directory: `.`
5. 배포 완료 후 URL 접속

`render.yaml`도 포함되어 있어 Blueprint 배포도 가능합니다.
